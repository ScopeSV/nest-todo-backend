## Description
Backend for todo redux testing apps, written in nest
## Installation

```bash
$ npm install
```
You will also need a google service account.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
